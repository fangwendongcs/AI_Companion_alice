# 本地免费 TTS 方案

当前默认语音使用浏览器/系统内置声线，保证不配置 API Key 也能出声。想要更接近中文动漫角色的效果，需要额外启动本地开源 TTS 服务，再由项目后端接入。

## 优先推荐

- CosyVoice：中文、英文、日文覆盖较好，支持低延迟与声音克隆，Apache-2.0。适合后续做高质量中文角色声线。
- Qwen3-TTS：支持中文、日文等 10 种语言，支持语音设计、情感和语速控制，Apache-2.0。适合后续做“用文字描述声线”的角色声音。
- ChatTTS：偏对话场景，中文/英文自然度较好。适合 AI 伙伴的日常对话，但实时性和授权边界需要再确认。
- Piper：轻量、离线、低延迟，适合兜底和弱设备。中文可用声线有限，动漫角色感弱于前几项。

## 当前策略

1. 默认：免费本机语音，保证交互一定有声音。
2. 高级云 TTS：MiniMax / OpenAI 可手动选择，失败时自动回退到免费本机语音。
3. 后续本地开源 TTS：建议优先接 CosyVoice 或 Qwen3-TTS 的本地 HTTP 服务，不需要浏览器保存 API Key。
